# آموزش راه اندازی رزبری پای پیکو ( Run raspberry pi pico )

سلام به همه توسعه دهندگان__
اگر به عنوان مقدمه بخواهم بیان کنم بنیاد رزبری پای مدل‌های کوچکی از برد توسعه خود را ارائه داده است که برخلاف باقی رزبری پای‌ها قابلیت نصب سیستم عامل را ندارند و مشابه میکروکنترلر‌ها عمل می‌کنند. زبان برنامه نویسی که برای این نوع از بردها می‌توان استفاده کرد میکروپایتون و سی می‌باشد.__
